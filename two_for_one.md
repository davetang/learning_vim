# README

There are many commands in Vim that compound two actions into a single keystroke.

| Compound command | Equivalent |
| -                | -          |
| `C`              | `c$`       |
| `s`              | `cl`       |
| `S`              | `^C`       |
| `I`              | `^i`       |
| `A`              | `$a`       |
| `o`              | `A<CR>`    |
| `O`              | `ko`       |
